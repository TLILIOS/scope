# iOS Senior Prep

Programme interactif de préparation aux entretiens iOS Senior.

**32 thèmes** · **4 semaines** · **Contenu généré par IA** · **Suivi de progression**

## Fonctionnalités

- Dashboard avec progression globale et par semaine
- 32 thèmes couvrant Swift, Concurrency, Architecture et System Design
- Contenu généré à la demande par Claude (concept, code Swift 6.2, questions d'entretien, pièges)
- Recherche instantanée sur tous les thèmes
- Progression sauvegardée localement (localStorage)
- Cache du contenu généré pour accès offline
- Interface responsive desktop + mobile
- Design dark premium

## Stack

Single-file HTML — zéro dépendance, zéro build.

- Vanilla JS
- Google Fonts (DM Sans, Source Serif 4, JetBrains Mono)
- Anthropic API (Claude Sonnet 4) pour la génération de contenu

## Déploiement

Hébergé sur GitHub Pages. Le fichier `index.html` est servi directement.
